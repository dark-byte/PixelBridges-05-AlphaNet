Build on Alph

Start building on Alph using Solidity and the tools you’re already familiar with: Remix, Truffle, and MetaMask. Fantom supports the EVM and you can deploy your dApps just like on Ethereum. Access to the documentation and connect with our developer community.

  • 01
    Open source

    Alph code is completely open source. Anyone can read it, check on the progress, comment on it, and help us build the ecosystem.

  • 02
    Launch customized blockchains, private & public

    Deploy blockchains that fit your own application needs. Build your own virtual machine and dictate exactly how the blockchain should operate.

  • 03
    Flexible, composable VMs

    Alph Virtual Machines (VMs) enable developers to easily launch blockchains with a wide array of application-specific features. The Ethereum Virtual Machine is fully running on Alph with more VMs to come.
